# Node.js, React and Socket.io chat app


### Project Setup
In the project directory:

Start Server
```
npm install
```
then,
```
npm run server
```


Start React Frontend
```
npm run clientinstall
```
then,
```
npm run client
```


Start both the Server and React frontend concurrently
```
npm run dev
```
